AI Recruiter Platform

An intelligent, multi-step career profile builder powered by Gemini 2.5 Flash. This application replaces traditional resume builders with an AI-driven conversational interface that suggests skills and structures work experience automatically.

🚀 Features

AI Skill Suggestions: Real-time professional skill recommendations as you type.

Experience Structuring: Automatically converts raw job descriptions into structured JSON (Role, Company, Achievements).

Recruiter Marketplace View: See exactly how your profile looks to potential employers.

Modern UI: Built with React, Tailwind CSS, and Lucide Icons.

🛠️ Tech Stack

Frontend: React.js

Styling: Tailwind CSS

Icons: Lucide React

LLM: Gemini 2.5 Flash (Google Generative AI)

📦 Getting Started

Prerequisites

Node.js (v18 or higher)

A Gemini API Key from Google AI Studio

Installation

Clone the repository

git clone [https://github.com/YOUR_USERNAME/ai-recruiter-platform.git](https://github.com/YOUR_USERNAME/ai-recruiter-platform.git)
cd ai-recruiter-platform


Install dependencies

npm install


Set up your API Key
Open App.jsx and replace the empty apiKey string with your Gemini API key:

const apiKey = "YOUR_ACTUAL_API_KEY";


Run the development server

npm run dev
