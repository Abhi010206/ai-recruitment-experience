# AI-Powered Recruitment Experience 🤖

## 🌟 Project Overview
This project is a React-based prototype designed to modernize the hiring process. It replaces traditional PDF resume uploads with an interactive AI-assisted profile builder, ensuring a more engaging experience for candidates and structured data for recruiters.

## ✨ Key Features
* **AI Conversational Input:** A dynamic chat interface that guides users through profile creation.
* **Structured Data Capture:** Automatically organizes candidate information into professional sections.
* **Recruiter Dashboard:** A clean, centralized interface for reviewing applicant profiles.

## 🛠️ Tech Stack
* **Frontend:** React.js
* **Styling:** CSS / Tailwind
* **Version Control:** Git & GitHub
